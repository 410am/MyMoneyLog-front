## 📕 설명

사용자의 소비 데이터를 기반으로
패턴 분석 + AI 인사이트를 제공하는 가계부 서비스입니다.

<br>

데이터를 시각화하고 소비 흐름을 분석하여
사용자가 자신의 소비 습관을 이해할 수 있도록 설계했습니다.

<br>

React, TypeScript를 사용하여 프론트를 구현하였고,
Spring Boot 기반 서버에서 JWT 인증 구조를 직접 구축했습니다.

또한 OpenAI API를 활용하여
사용자의 소비 데이터를 자연어 인사이트로 변환하는 기능을 구현했습니다.
